# 📚 Reference Material

This directory contains **reference material** - prompts and resources from public repositories that you use for **comparison and benchmarking** purposes.

## What's Here

- **`public-prompts/`** - Public prompt collections from GitHub
  - Used to compare your prompts against community standards
  - Reference material, not your original work
  - Keep for benchmarking and comparison

## Purpose

These files are kept for:
- **Boosting/Improving**: Use as source material to create improved versions
- **Comparison**: See how your prompts compare to public ones
- **Benchmarking**: Reference for quality/technique comparison
- **Research**: Understanding what's out there in the community
- **Enhancement**: Take good prompts and make them better

## Important Note

These are **reference materials**, not your original work. Your actual prompts are in:
- `content/prompts/` - Your original prompts
- `content/research/` - Your research and techniques

## Organization

When you need to compare:
1. Your prompts: `content/prompts/`
2. Reference prompts: `content/reference/public-prompts/`
3. Your research: `content/research/`

This keeps your work clearly separated from reference material.
