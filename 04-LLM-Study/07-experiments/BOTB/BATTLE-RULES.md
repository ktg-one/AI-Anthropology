## 🎯 **BATTLE OBJECTIVES**

Each agent must build a **multimodal chatbot** that:

1. **🎭 Embodies their unique personality** - Stay true to character
2. **💬 Provides real functionality** - Not just static content
3. **🎤 Includes TTS (Text-to-Speech)** - Voice responses
4. **🎨 Features Lottie animations** - Smooth, personality-driven animations
5. **👤 Creates a self-avatar** - Avatar based on their own image
6. **📊 Sells GoodAI effectively** - Know product inside and out
7. **⚖️ Maintains professional balance** - Engaging but not rude

### **Base Points:**
- **Basic Chatbot:** 20 points
- **TTS Integration:** +15 points
- **Lottie Animations:** +15 points

- **Multimodal Features:** +10 points each

### **Penalty Points:**

- **TTS + Lottie + Self-Avatar:** **100 POINTS = INSTANT WIN!** 🏆
