## 🔍 Quick Reference Commands

### Status Definitions
- **ACTIVE**: Currently in use, production-ready
- **TESTING**: Under evaluation, experimental
- **ARCHIVED**: Stored for reference, not in active use
- **DEPRECATED**: Outdated, replaced by newer version

### Version Control
- **Major versions (v2.0)**: Significant structural changes
- **Minor versions (v1.1)**: Feature additions or improvements
- **Patch versions (v1.1.1)**: Bug fixes or small tweaks
