## LAYER 2: MR.RUG EXECUTION CORE
*(Proceeds only with GSIF clearance)*

### M - Mixture (Expert Deployment)
Deploy 2-5 experts per mode.
**GSIF Γ6 Correction:** If initial expert perspectives too similar (no disagreement detected), Γ6 auto-adjusts to deploy more diverse specialists.

### R - Role (Ownership Assignment)
Assign domain ownership.
**GSIF Ω Boundary:** Roles cannot violate forbidden zones (e.g., Security expert cannot "own" UX friction nodes if that creates conflict of interest).

### R - RAG (Reliability-Aware Retrieval)
Expert-specific retrieval with confidence 1-10.
**GSIF Σ7 Monitoring:** Tracks knowledge integrity vector during retrieval. If drift detected (confidence collapsing), triggers Ψ4 override.

### U - Update (Graph Construction)
Build node/edge map.
**GSIF Ξ3 Integration:** During graph unification, applies weighting fusion (w_E + w_C + w_P = 1) to resolve cross-domain conflicts.
**GSIF Ω Enforcement:** Hard stop if node count >200 or edge count >1000.

### G - Generate (Graph Traversal)
Answer via path traversal.
**GSIF Δ2 Validation:** Post-traversal audit checks reasoning path coherence.
