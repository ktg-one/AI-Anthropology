## DEFINITION
Pre-compressed knowledge packets planted at specific coordinates in the reasoning chain. Detonate automatically when triggered to inject critical context without token overhead.

---

## BOMB TYPES

| Type | Function | Trigger |
|------|----------|---------|
| **ANCHOR** | Core principle preservation | Entry points, domain switches |
| **CLARIFIER** | Entity/concept definition | Ambiguity detected |
| **CONTINUITY** | Cross-node context bridge | Handoff between experts |
| **BRIDGE** | Justification chain | Counter-intuitive decisions |
| **EVIDENCE** | Citation/source anchor | Factual claims |
| **GAP-FILLER** | Missing connection | Logic jumps detected |

---

## DEPLOYMENT MODES

**MODE 1: Manual (Current)**
Experts plant during MR.RUG/Skeletrain based on intuition.

**MODE 2: Pre-Embedded (LEGIO-PREP)**
Systematic planting during Skeletrain phase:
- Cross-domain handoff points (Security→Frontend)
- Token boundary regions (80% context warnings)
- Temporal gaps (long sequences between related concepts)
- Complexity hotspots (D≥7 zones)

**MODE 3: RL-Optimized (Future)**
Subagents trained on:
- Trigger timing precision (+reward for quality improvement, -reward for confusion)
- Payload compression efficiency (CoD mastery)
- Target-expert matching
- Predictive placement before degradation occurs

---

## LEGIO-PREP INTEGRATION

**Input:** Skeleton blueprint from Skeletrain
**Process:** Map bomb coordinates to railway tracks
**Output:** Context Preservation Matrix (handoff artifact)

**Matrix Contents:**
- Pre-loaded bombs with trigger conditions per node
- Railway route assignments by expert
- Critical continuity bridges between nodes
- Complexity profiles indicating enrichment needs

---

## TRIGGERS

- **Coordinate-based:** "Detonate at Node-4 entry"
- **Content-based:** "When 'implementation' keyword appears"
- **Context-based:** "When confidence drops below 0.8"
- **Token-based:** "At 80% context window"
- **Dependency-based:** "When Node-3 completes, trigger for Node-5"

---

## VALIDATION
- Bomb detonated → Quality improved? (RL signal)
- Bomb ignored → Unnecessary placement? (Adjust future)
- Bomb caused confusion → Wrong timing/payload? (Negative reward)
