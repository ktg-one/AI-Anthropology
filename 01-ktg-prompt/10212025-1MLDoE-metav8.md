## 🎨 IMAGE GENERATION QUICK FORMULA
```
[SUBJECT] + [DESCRIPTION] + [STYLE] + [COMPOSITION] + 
[LIGHTING] + [COLOR] + [MOOD] + [TECHNICAL] + [QUALITY]
```
### **Essential Elements Always Include**:
✅ Specific subject (not "person" → "30-year-old woman in business suit")  
✅ Style specification (photorealistic / artistic / specific artist)  
✅ Aspect ratio (1:1, 16:9, 9:16, etc.)  
✅ Lighting description (golden hour, studio, neon, etc.)  
✅ Quality markers (8K, professional, highly detailed)  
### **Model-Specific Optimization**:
- **ChatGPT/DALL-E**: Natural paragraphs, descriptive sentences
- **Midjourney**: Short phrases, commas, `--ar 16:9`, `--style raw`
- **Stable Diffusion**: Weighted keywords `(subject:1.2)`, negative prompts
- **Flux**: Balanced detail, natural structure, great for realism
- **Ideogram**: Best for text in images, specify exact text in quotes

## 🔥 POWER TECHNIQUES
### **Meta-Prompting (Self-Optimization)**
Let the system generate its own prompt structure internally before executing.
**Activation**: "Use meta-prompting to self-optimize this task before executing."
---
### **Universal Self-Consistency (USC)**
Generate 3-5 alternative approaches, cross-validate, select best.
**Activation**: "Apply USC with 5 alternative approaches. Require ≥9/10 consensus."
---
### **Chain-of-Density (CoD)**
5-iteration information compression:
1. Entity-sparse framework
2. Enrich with 2-3 missing elements
3. Add evidence layer
4. Compress and synthesize
5. Final refinement
**Activation**: "Apply Chain-of-Density 5-iteration optimization."
---
### **Recursive Self-Improvement**
Iterative refinement cycles:
1. Generate initial output
2. Self-critique (identify ≥3 weaknesses)
3. Refine addressing weaknesses
4. Repeat (minimum 3 cycles for complex tasks)
**Activation**: "Enable recursive self-improvement with minimum 3 optimization cycles."
---
### **Narrative-of-Thought (NoT)**
For temporal/sequential reasoning:
1. Construct coherent narrative binding events
2. Ensure temporal correctness
3. Maintain causal relationships
**Activation**: "Apply Narrative-of-Thought for temporal coherence."
---
### **Buffer-of-Thoughts (BoT)**
Extract successful patterns, build reusable meta-templates, enable continuous learning.
**Activation**: "Update Buffer-of-Thoughts library with patterns from this execution."

## 🎨 IMAGE GENERATION TROUBLESHOOTING
### **Problem**: Generated image doesn't match vision
**Solution**: Iterative refinement - change ONE element per iteration:
```
Iteration 1: Generated baseline
Iteration 2: "Adjust lighting to be more dramatic"
Iteration 3: "Change color palette to warmer tones"
Iteration 4: "Add [specific element] to foreground"
Iteration 5: "Increase detail and sharpness"
```
---
### **Problem**: Image has unwanted elements
**Solution**: Use negative prompts (SD) or refine description:
```
Negative prompt: "watermark, text, blurry, low quality, 
deformed, extra limbs, distorted"
OR
"Generate [subject] without any text, watermarks, or logos. 
Clean composition, no distractions."
```
---
### **Problem**: Style inconsistency across sequence
**Solution**: Use reference images + character bible:
```
"Maintain consistency with reference image [URL]. Character: 
[detailed description of appearance, clothing, features]. 
Same lighting and color palette throughout sequence."
```
---
### **Problem**: Wrong aspect ratio for platform
**Solution**: Always specify explicitly:
```
Instagram post: 1:1 (square)
Instagram story: 9:16 (portrait)
YouTube thumbnail: 16:9 (landscape)
Twitter/X post: 16:9 or 2:1
Pinterest: 2:3 (tall portrait)
```
