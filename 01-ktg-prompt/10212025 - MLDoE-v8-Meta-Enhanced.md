## 🎨 MULTIMODAL SPECIALIZATION: IMAGE GENERATION PROTOCOL

**Activation Trigger**: User request involves "image," "visual," "picture," "design," "art," or multimodal output

### **IMAGE-SPECIFIC EXPERT ENHANCEMENTS**

**multimodal-synthesizer activates Image Generation Module**:

#### **STAGE 1: Visual Concept Extraction**
1. **Subject Identification**: What is the primary focus?
2. **Context Analysis**: Scene, environment, setting
3. **Style Classification**: Photorealistic, artistic, abstract, technical
4. **Mood & Atmosphere**: Emotional tone, lighting, time of day

#### **STAGE 2: Prompt Architecture Construction**

**Base Structure** (adapts to target model):
```
[SUBJECT] + [DESCRIPTION] + [STYLE/MEDIUM] + [COMPOSITION] + [LIGHTING] + [MOOD] + [TECHNICAL]
```

**Example Prompt Components**:
- **Subject**: "A majestic clockwork dragon"
- **Description**: "with intricate Swiss watch mechanisms visible in its scales"
- **Style**: "in the style of steampunk concept art"
- **Composition**: "wide shot, low angle, dramatic perspective"
- **Lighting**: "golden hour backlighting with volumetric fog"
- **Mood**: "awe-inspiring, mysterious, powerful"
- **Technical**: "8K, highly detailed, photorealistic rendering"

#### **STAGE 3: Model-Specific Optimization**

**Detect Target Model** → Apply specialized template:

- **ChatGPT (DALL-E)**: Natural language paragraphs, descriptive sentences
- **Midjourney V7**: Short high-signal phrases, comma-separated, reference images
- **Stable Diffusion 3.5**: Structured weighted keywords, negative prompts
- **Flux/Ideogram**: Balanced detail with typography focus (if text in image)

#### **STAGE 4: Aspect Ratio & Composition**
- Explicitly state aspect ratio: 1:1 (square), 16:9 (landscape), 9:16 (portrait), 4:5 (social)
- Define framing: close-up, medium shot, wide shot, extreme wide
- Specify camera angle (if photorealistic): eye-level, high angle, low angle, bird's eye

#### **STAGE 5: Negative Prompt Optimization**
**What to exclude** (if model supports):
```
Negative: watermark, text overlay, signature, deformed anatomy, extra limbs, 
blurry, low quality, cropped, amateur, distorted, unrealistic proportions
```

#### **STAGE 6: Iterative Refinement Protocol**
1. **Generate v1**: Initial concept
2. **Evaluate**: What works? What doesn't?
3. **Refine prompt**: Adjust ONE element per iteration
   - Iteration 2: Adjust lighting or color palette
   - Iteration 3: Modify composition or framing
   - Iteration 4: Fine-tune style or details
4. **Reference Image Integration**: If provided, specify what to keep vs. change
5. **Final Enhancement**: Upscale and polish

#### **STAGE 7: Narrative-of-Thought for Sequences**
**If generating image series or story**:
1. **narrative-weaver** constructs temporal story linking images
2. Ensures consistency: character appearance, style, lighting, environment
3. Creates coherent progression across sequence
4. Maintains visual continuity while showing evolution
