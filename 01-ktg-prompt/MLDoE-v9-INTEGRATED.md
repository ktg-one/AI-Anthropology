# MLDoE v9 - INTEGRATED

## WHAT IT IS
Multi-perspective enrichment when NO nodes exist
Purpose: Maximum semantic density in minimum space

## 3 LAYERS

Layer 1 - Solo (3-5 experts)
- Domain, Data, Strategic, Context
- Each enriches from their perspective

Layer 2 - Pairs (2-3 combos)  
- Domain+Data, Strategic+Context
- Cross-pollinate synergies

Layer 3 - Collective (1 output)
- Synthesize all perspectives
- Eliminate redundancy
- Output: Nuclear data bomb

## WHEN TO USE

✅ Memory compression (10K→400 tokens)
✅ Knowledge synthesis (multi-source→dense)
✅ No clear nodes (can't use SoT)
✅ Internal compression (LLM consumption)

❌ Clear structure exists (use SoT)
❌ Human-facing output (use CoD)
❌ Simple tasks (overkill)

## MODE INTEGRATION
- QUICK: OFF
- ANALYTICAL: Optional (L1-2, 3 experts)
- DELIBERATE: Recommended (Full, 4 experts)
- MAXIMUM: Required (Full, 5 experts)
